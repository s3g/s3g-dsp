Ambi Stereo Decoder

s3g Ambi Stereo Decoder decodes ACN/SN3D ambisonic input to a true stereo output using virtual speaker fields and practical stereo pickup models. It is for loudspeaker-style stereo monitoring or stereo renders, not headphone binaural playback.

input1OA..7OA
formatACN/SN3D
outputtrue stereo
routingREAPER pins

Model

The decoder projects the ambisonic signal onto a virtual speaker field, then derives stereo from that field using a selected pickup method. This makes the signal path readable: first choose how the ambisonic field is sampled, then choose how the stereo image is derived.

Available virtual fields include quad, 8-channel cube, 12-channel dodeca, 24-channel dome, and 32-channel sphere. These are internal sampling fields, not necessarily the user's physical speaker layout.

Pickup Methods

  • XY cardioid: coincident angled cardioid-style pickup.
  • ORTF-style: spaced and angled cardioid-style pickup. It suggests ORTF behavior but is not a measured physical microphone model.
  • MS cardioid: mid/side pickup, displayed as M, S+, and S-.
  • Blumlein: crossed figure-8-style pickup with dashed negative lobes in the display.
  • Spaced omni: A/B-style pickup. The AB control changes capsule spacing and is visualized in the field.
  • Dual shotgun, Wide cardioid, Supercardioid XY, Hypercardioid XY, and Height focus: experimental pickup shapes for comparison.

Controls

  • ORD: ambisonic order, from 1OA to 7OA.
  • FIELD: internal virtual field used for projection.
  • MTHD: stereo pickup method.
  • WDTH, ANG, and ROT: stereo width, pickup angle, and listening rotation.
  • DIR, REAR, HGT, and DIF: directivity, rear rejection, height fold, and diffuse blend.
  • WGT and AGN: ambisonic order weighting and output autogain.
  • AB: spacing for Spaced omni.
  • BASS: optional bass mono fold below the selected frequency.
  • M-EL: microphone elevation for pickup models that use vertical orientation.
  • OUT: final output gain.

REAPER Routing

This plugin exposes a true stereo output bus. In REAPER, a multichannel track can still preserve audio on higher track channels that are not mapped through the plugin output. If a following meter shows channels above 2 after the decoder, open REAPER's plugin pin connector and enable Zero out unmapped output channels.

This appears to be a host pin-routing behavior with true stereo CLAP outputs on wider REAPER tracks. Fixed-width s3g-dsp plugins can clear their own unused lanes because those lanes remain part of the plugin output bus. A true stereo decoder cannot reliably clear channels that REAPER routes around the plugin.

For headphone-style monitoring, use Ambi Head Decoder. For speaker layout playback, use Ambi Speaker Decoder 64.
