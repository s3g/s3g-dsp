s3g-dsp

CLAP audio plugins for multichannel work in REAPER. s3g-dsp is a sibling project to the s3g-mc multichannel REAPER package and the s3g-max Max/MSP externals package. It can be used with those projects or on its own.

The current focus is predictable multichannel routing, compact automation, and clear multichannel control models. Some plugins use topology; others use simpler relationship controls when that better fits the sound.

The supported target is macOS + REAPER. Other operating systems and DAWs are not support targets at this stage.

Start Here

  • Install pre-release plugins if you want to try the current macOS CLAP bundle.
  • Build from source if you want to compile locally.
  • Multichannel covers effects, fold-down, metering, direct panning, bus mixing, and speaker utilities for ordinary channel lanes.
  • Ambisonics separates encoders, decoders, effects, and utilities for ACN/SN3D workflows.
  • Instruments covers loaded-audio and synthesis CLAP instruments.
  • References collects the external sources that materially informed the package.

Status

This is a pre-release project. Plugin names, parameters, included bundles, and saved states may change. The current macOS package installs 81 CLAP bundles, including fixed-width variants for several effects and routing tools.

For the current installable bundle list, use Installing Pre-release Plugins. The front page stays short so it remains useful as the package grows.

Design

  • Reusable DSP lives in dsp/; CLAP wrappers live in plugins/.
  • Fixed-width plugins are used where REAPER pin routing needs to stay predictable.
  • Many plugins expose compact relationship controls instead of one automation lane for every channel.
  • Multichannel, Ambisonics, and Instruments are the main documentation groups; each separates effects from routing or source-generation workflows.
  • The central References page groups external sources by method and uses the same author–date style as s3g-mc.

Topology Layer

Topology is used only where point and network control helps the sound. A patch matrix selects lanes, those lanes become points, and each topology plugin maps the point network to its own parameters.

The Topology Effects section begins with the shared Topology Framework, then follows it with the delay, wave-geometry, and spectral mappings.

Related Projects

s3g-mc is the main REAPER package. s3g-dsp is a sibling plugin project that can be used alongside s3g-mc or independently in matching macOS + REAPER sessions.

s3g-max is the related Max/MSP package for .mxo externals, help patches, V8UI displays, and Max package workflows.

s3g-rnbo-clap is a separate wrapper lab for adapting RNBO C++ exports as CLAP plugins in the same visual and release style.
