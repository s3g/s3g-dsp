Ambisonic Decoders

Decoders turn an ACN/SN3D field into speaker, object, subwoofer, stereo, or headphone feeds. Match the decoder order to the encoded material and make its output bus the final spatial rendering stage.

Speaker and Object Rendering

  • Ambi Speaker Decoder: decodes first-through-seventh-order material to built-in or custom speaker layouts.
  • Object, Adaptive, and Sub variants: focus the decode on object directions, adapt the rendering behavior, or derive low-frequency feeds while retaining the shared layout model.

Stereo and Headphone Monitoring

  • Ambi Stereo Decoder: rotation-aware virtual-speaker decode to a true stereo output bus.
  • Ambi Head Decoder: binaural headphone rendering with listener orientation and room controls.

Host Routing

In REAPER, set the track wide enough for the incoming order before inserting the decoder. Stereo and headphone variants expose a true two-channel output; enable Zero out unmapped output channels in the pin connector if REAPER passes higher track channels around the plugin.
