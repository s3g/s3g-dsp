Distributed-output Effects

Distributed-output effects accept a stereo source and generate sixteen discrete output lanes. They create multichannel time and event structures without encoding the result as an ambisonic field.

Plugins

  • Shard Scatter: cuts a live stereo buffer into grain-like shards and distributes those events across sixteen outputs.
  • Orbit Delay: turns stereo input into a sixteen-output orbiting delay cloud with transport-stop de-click behavior.
  • Cascade Taps: builds a sixteen-output stepped tap ring with a soft handoff control for changing cascade structure.

Routing

Set the REAPER track to at least sixteen channels and confirm the plugin pin mapping before auditioning. The first two track channels feed the input bus; the effect publishes sixteen discrete outputs for downstream routing, mixing, or direct speaker work.

Discrete, Not Ambisonic

These outputs are ordinary lanes. Use Direct Panning or Bus Mixing / Routing to place or combine them. Do not send them directly to an ambisonic decoder unless they have first passed through an Ambisonic Encoder.
