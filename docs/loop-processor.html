Loop Processor

s3g Loop Processor 8ch is a CLAP instrument plugin for loaded audio loops. It does not process track input; it plays a selected sound file across eight output lanes.

The current model is lane based rather than topology based. A loaded mono, stereo, or multichannel file is repeated across the eight outputs, and each lane has its own playhead, phase, rate relationship, and shared loop region.

Workflow

  • Insert s3g Loop Processor 8ch on an 8-channel REAPER track.
  • Click LOAD and choose an audio file.
  • Use the transport icons to play, pause/resume, or stop and return the lane playheads to the start.
  • Use the output lane strip to enable or mute individual lanes.
  • Shape lane timing with the relationship and loop sections.
  • Use SYNC to realign lane playheads after editing loop timing.

If the loaded file has fewer than eight channels, source channels repeat across the eight output lanes. For example, a stereo file maps odd lanes from channel 1 and even lanes from channel 2.

Display

Waveform

Shows the loaded channel with the strongest activity. Loop start and length are drawn as markers over the waveform.

Lane Playheads

Thin overlay cursors show where enabled lanes are inside the loop region. The relationship panel shows lane-rate differences as horizontal bars.

Lane Strip

The compact output strip enables or mutes individual output lanes without invoking the topology patch matrix model.

Engine

Control Meaning
RATEBase playback rate for all lanes.
XFDLoop seam crossfade as a percentage of the active loop length, capped at 30%. This keeps the seam proportional as LEN changes.
DUCKOptional seam ducking inside the crossfade window.
OUTOutput gain after lane playback.
STRTLoop start position inside the loaded file.
LENLoop length. The default is the full file.

Relationships

Relationship controls change the timing relationship between lanes. They are global controls, but the plugin derives per-lane behavior internally.

Control Meaning
SPRDBipolar rate spread. Positive values fan lane rates outward; negative values mirror the relationship.
DRFTBipolar slow drift amount. Negative values invert the drift direction.
CTRRelationship center. Moving it changes which lanes behave like the middle of the spread.
GLDGlide time for relationship changes, useful for automation and large jumps.

Transport

  • Play starts or resumes lane playback.
  • Pause holds the lane positions. Clicking pause again resumes playback.
  • Stop pauses and returns all lane playheads to the start.

Current Notes

Loop Processor is intentionally separate from the topology framework. It borrows the idea of compact relationship controls, but its visual and DSP model is built around loop lanes, playheads, region motion, and seamless loop playback.
