3OAFX Effects

The 3OAFX effects are single-purpose ambisonic processors for third-order ACN/SN3D material. Each plugin decodes the input to the fixed 24-point virtual speaker layer, applies one spatial effect with a return mask, then encodes back to 3OA.

This replaces the earlier rack experiment. The rack source remains in the repository as a development checkpoint, but current builds favor separate effects so there is no multi-effect summing inside one plugin.

input3OA
field24 points
output3OA
hostREAPER

Plugins

spatial delay s3g 3OAFX Delay Delay effect returned through one AED mask. Use Time, Feedback, Mix, Dry, Output, and the mask position/width controls.
spatial pitch s3g 3OAFX Pitch Pitch-shifted return through one AED mask. Use Semitones, Mix, Dry, Output, and the mask controls.
spatial filter s3g 3OAFX Filter Filtered return through one AED mask. Tone moves from darker body to brighter, more separated material.
spatial gain s3g 3OAFX Gain Masked level shaping. Gain above 1.0 boosts the area, below 1.0 reduces it, and 0.0 cuts a hole in the sound field.

Signal Flow

3OA input
-> decode to 24 virtual speaker points
-> dry copy path
-> one effect return and one AED return mask
-> mixer
-> encode back to 3OA output

The dry copy does not pass through the effect. It is decoded to the same 24-point layer so the masked return can replace, duck, boost, or cut a spatial area before the final 3OA encode.

Why Separate Effects

A single plugin with several internal effects can create useful sounds, but it also creates level and masking interactions that are hard to diagnose. The separate-effect design keeps each processor predictable. Stack multiple plugins in REAPER when several 3OAFX effects are needed.

For parallel combinations, use separate REAPER routing or future utility plugins rather than a hidden internal rack mixer.

Shared Controls

ControlMeaning
Azimuth / ElevationCenter of the return mask in the 24-point virtual speaker field.
WidthHow wide the affected area is.
DryDecoded dry copy level in the mixer.
MixWet return level for Delay, Pitch, and Filter.
OutputFinal level after the 24-point mixer and 3OA encode.
GainSpatial level amount for 3OAFX Gain. 1.0 is neutral.

REAPER Routing

These plugins expose a 16-channel 3OA input and output bus. Put them on a 16-channel 3OA track. They do not require a 72-channel send/return chain.

Use them before a speaker decoder, stereo decoder, head decoder, or energy visualizer. For the broader REAPER 3OAFX workflow, see the s3g-mc 3OAFX process guide.
