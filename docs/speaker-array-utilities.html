Speaker Array Utilities

Speaker-array utilities sit after direct panning or decoding. They prepare discrete speaker feeds for the physical system rather than moving sources or changing the artistic mix.

Sub Crossover

Sub Crossover derives low-frequency sends from a Layout Panner speaker array and supports as many as eight subwoofer channels. Use it when bass management belongs to the rendered speaker bus rather than the source or ambisonic field.

Array Calibration

  • Array HPF 16/26/32/64: post-render high-pass filtering for main speakers.
  • Array Delay 16/26/32/64: independent alignment delay for every speaker feed.
  • Array Trim 16/26/32/64: per-channel gain, mute, and polarity correction.

Recommended Order

direct panner or ambisonic decoder
-> Sub Crossover / Array HPF
-> Array Delay
-> Array Trim
-> hardware outputs

Choose the fixed-width variant that matches the rendered array. Keep artistic automation upstream; calibration settings should normally remain stable for a given room and output interface.
