Multichannel Meter 64

s3g Multichannel Meter 64 is a fixed 64-in/64-out passthrough meter for checking REAPER multichannel routing. It does not change the signal. Use WIDTH to show only the channels you care about while keeping the track bus predictable.

Views

  • GRID: channel cells with RMS fill and peak tick. This is the direct routing/debug view.
  • FIELD: channels drawn as a spatial layout-style field. Choose common speaker layouts such as quad, quad plus overhead, cube, dome, 3OAFX 24, dodeca, octophonic ring, ring 12/16, or double ring 16/20. Marker size follows loudness, VU-style color follows level, and red marks very high peaks.
  • HEAT: energy over time. Each lane scrolls horizontally so bursts, dropouts, and sustained channel activity are easier to see.

Field Camera

The field view uses the same inspection controls as the spatial plugins: TOP, SIDE, 3/4, plus camera zoom. Drag in the field to rotate the 3/4 view without changing audio or routing.

Routing

The plugin exposes one 64-channel input bus and one 64-channel output bus. Audio passes through channel-for-channel, and unused output lanes are cleared. If a project uses a smaller working width, leave the REAPER track width as needed and set WIDTH to the number of channels you want to inspect.

Use Cases

  • Confirm that a 24-channel or 25-channel chain is only using the intended lanes.
  • Check whether a fold-down or panner leaves unexpected audio on higher channels.
  • Watch a multichannel texture as an energy pattern instead of reading one peak meter at a time.
