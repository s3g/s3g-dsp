3OAFX Insert Workflow

The 3OAFX tests target the 24-channel virtual speaker layer used by the s3g-mc workflow. This page is a work-in-progress note for the CLAP boundary plugins.

For the broader ambisonics plugin group, see the 3OAFX overview. For point-source encoding and AED automation, see 3OAFX Point Encoder. For speaker playback and virtual layout testing, see 3OAFX Speaker Decoder. For direct source-to-speaker routing, see Layout Panner.

Signal Path

3OA source
-> s3g 3OAFX Send Decoder
-> 24-channel insert plugin
-> s3g 3OAFX Return Encoder
-> 3OA output

s3g 3OAFX Send Decoder decodes a 3OA ACN/SN3D input to the fixed 24-point virtual speaker layer and outputs a 72-channel bus.

  • 1-24: wet/effect lane.
  • 25-48: decoded dry copy.
  • 49-72: mask signal.

s3g 3OAFX Return Encoder reads the same 72-channel bus, applies the return mask/mixer, and encodes the result back to 3OA ACN/SN3D.

The mask signal on 49-72 syncs the two boundary plugins. REAPER parameter links are still recommended so the Return Encoder GUI follows the Send Decoder.

REAPER Parameter Links

After inserting the boundary pair, link the Return Encoder direction parameters to the Send Decoder.

  1. Put the plugins on a 72-channel REAPER track in this order: s3g 3OAFX Send Decoder, a 24-channel insert effect, then s3g 3OAFX Return Encoder.
  2. Open s3g 3OAFX Return Encoder.
  3. Use REAPER's Param menu for Azimuth, then choose Parameter modulation/MIDI link.
  4. Enable parameter linking from another FX parameter.
  5. Set the source FX to s3g 3OAFX Send Decoder.
  6. Set the source parameter to Azimuth.
  7. Use a neutral 1:1 scale and offset.
  8. Repeat the same link for Elevation.

Useful optional links include Width, Focus, and Rear Reject.

When these links are active, moving the Send Decoder direction controls should also move the Return Encoder GUI. If REAPER still has an older bundle loaded, reload the plugin instances or restart REAPER.

Plugin Shape

s3g 24ch Passthrough Test should only change gain. It exists to confirm that REAPER preserves 24-channel audio correctly.

s3g Delay Processor 24ch is the current wide insert effect for testing this boundary.

3OAFX-specific processors should stay 24-in/24-out when they sit directly on the virtual speaker layer. There, channel count and speaker order are part of the instrument.
