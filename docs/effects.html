Effects

Effects process incoming REAPER track audio without assuming an ambisonic format. Ambisonics-specific plugins now live in the Ambisonics section.

Effect Pages

  • Delay Processor: main topology-driven delay effect and the best starting point for topology controls.
  • Macro Delay: compact 8-channel delay macro with relationship controls.
  • Macro Pitch: compact 8-channel pitch macro using the shared Macro control layout.
  • MC to Stereo Autogain: multichannel-to-stereo audition fold-down.
  • MC to Quad Autogain: multichannel-to-quad audition fold-down.
  • Spectral Spray 2ch / 8ch: FFT-bin scatter, smear, hold, freeze, and feedback effects.
  • Shard Scatter: 2-in/16-out grain-shard spatial scatter effect.
  • Orbit Delay: 2-in/16-out orbiting delay effect.
  • Cascade Taps: 2-in/16-out stepped tap-ring effect with cascade handoff softening.
  • Group Matrix 64: general 64-channel group matrix with 4ch, 8ch, and 16ch group modes.
  • Multichannel Meter 64: fixed-width multichannel passthrough meter.

Processors

topology delay Delay Processor Main topology effect. Covers 8ch and 24ch lane processors, REAPER pin routing, the internal patch matrix, topology mapping, delay engine controls, and the current GUI.
FFT scatter Spectral Spray 2ch / 8ch FFT-bin scatter, drift, smear, hold, freeze, holes, phase blur, feedback, and frequency-window effects.
grain scatter Shard Scatter 2-in/16-out grain-shard scatter with density, guard time, scatter range, pitch spread, rotation, width, and feedback safeguards.
orbiting delay Orbit Delay 2-in/16-out orbiting delay field with spread, focus, feedback, damping, and transport-stop de-click behavior.
tap cascade Cascade Taps 2-in/16-out stepped tap-ring processor with base/step timing, decay, damping, direction, and a SOFT handoff control.

Macros

delay macro Macro Delay 8-channel and 24-channel delay macros with shared engine controls and lane relationship controls such as spread, deviation, skew, center, and glide.
pitch macro Macro Pitch 8-channel and 24-channel pitch macros using the same Macro relationship panel style as Macro Delay.

Utilities

monitoring utility MC to Stereo Autogain 128-channel input to true stereo fold-down for auditioning multichannel tracks. monitoring utility MC to Quad Autogain 128-channel input to quad output fold-down with output order L, R, RB, LB. metering utility Multichannel Meter 64 Fixed 64-in/64-out passthrough meter with grid, spatial layout, and energy-over-time views.
bus utility Group Matrix 64 General 64-channel group matrix for 4ch, 8ch, or 16ch blocks. For 3OA buses, use the Ambi Group Matrix variants in Ambisonics.
routing check Passthrough Test 24ch Fixed-width gain and routing check for REAPER and CLAP bundle validation.
