Ambi Bus Processors

The Ambi Group processors work on several complete third-order ambisonic feeds at once. Each feed is treated as one locked 16-channel ACN/SN3D block, so routing, rotation, and depth shaping do not break the internal channel relationships of the ambisonic signal.

Plugins

group routing Ambi Group Matrix 64 / 128 Matrix mixing between four or eight 3OA feeds. Manual matrix levels set the ceiling; flow controls can create active movement inside that ceiling.
group motion Ambi Group Rotate 64 / 128 Rotates each 3OA group coherently. Global yaw, pitch, and roll define the base motion; spread, tilt, and twist distribute offsets across groups.
group depth Ambi Group Depth 64 / 128 Shapes apparent depth across 3OA groups using bipolar depth, spread, focus, air, low-body, order-width, and output controls.
general matrix Group Matrix 32 / 64 General group matrix mixers for non-ambisonic blocks. The 32-channel version supports 2ch, 4ch, and 8ch groups; the 64-channel version supports 4ch, 8ch, and 16ch groups.

Matrix Model

The matrix processors route one complete group into another complete group. In the Ambi versions, a crosspoint routes all 16 channels of a 3OA feed together. This keeps the spatial encoding intact while still allowing a bus-level mix between feeds.

  • SHAPE: generated gain pattern. Current choices include flow, pulse, chase, swirl, scatter, and hold.
  • MODE: free-running or transport-synced timing for animated flow.
  • FLOW: how strongly the generated flow layer enters the matrix.
  • SPRD, MOT, and related flow controls: how wide, animated, or patterned the active matrix layer becomes.
  • Gray matrix cells show the manual ceiling. Green activity shows the current flow inside that ceiling.

Rotate Model

Rotate applies a coherent ambisonic rotation to each group. It is useful when several 3OA beds or stems need related movement without building separate automation for every feed.

  • YAW: horizontal rotation for all groups.
  • PIT: vertical pitch/elevation rotation for all groups.
  • ROL: roll around the front axis.
  • SPRD: yaw offset spread across groups.
  • TILT: pitch offset distribution across groups.
  • TWST: roll offset distribution across groups.
  • WID: order-width trim for the rotated groups.

Depth Model

Depth is a bipolar bus-shaping tool. Center is neutral. Positive values push groups farther and softer by reducing higher-order spatial detail and applying optional air damping. Negative values push toward a nearer, more present image.

  • DEP: near/presence to far/soft depth transform.
  • SPRD: bipolar distribution of depth offsets across groups.
  • FOC: diffuse to focused order behavior.
  • AIR: negative values add presence; positive values add distance damping.
  • LOW: low-body compensation or reduction.
  • WID: order-width trim.
  • OUT: final output gain.

Routing

The 64-channel versions are intended for four 3OA groups. The 128-channel versions are intended for eight 3OA groups. In REAPER, set the track channel count before inserting the plugin so the pin connector matches the intended width.

Use Group Matrix 32 for stereo, quad, or octo groups. Use Group Matrix 64 for larger non-ambisonic 4ch, 8ch, or 16ch blocks. Both use the same matrix and flow model without assuming 3OA format.
