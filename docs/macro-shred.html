Macro Shred

s3g Macro Shred is a pressure, wavefolding, and bounded resonant-feedback processor designed for voice and microphone input. The same mono DSP kernel is available as Mono, 8-channel, and 24-channel CLAP plugins.

Versions and Routing

  • s3g Macro Shred Mono: one input and one output channel with the core voice controls.
  • s3g Macro Shred 8ch: eight independent kernels with shared controls and lane relationships.
  • s3g Macro Shred 24ch: twenty-four independent kernels with the same multichannel control model.

Set the REAPER track channel count before inserting an 8-channel or 24-channel version. Each input lane is processed independently; the plugin does not mix or ambisonically encode its channels.

Voice Controls

ControlMeaning
INPUTApplies gain before envelope analysis, pressure, and the feedback loop.

Centroid Tuning

TUNE uses a lightweight four-band log-frequency estimate rather than an FFT. Confidence gating prevents very quiet input from steering the loop, and smoothing keeps consonants from producing abrupt frequency jumps. At 0%, feedback follows COLOR alone. At 100%, the input centroid provides the main tuning target while COLOR remains an octave offset.

The LOOP readout shows the current feedback frequency beside its activity meter. In the multichannel versions it shows the geometric center of the active lane frequencies.

Multichannel Relationships

The 8-channel and 24-channel versions add the same relationship controls used by Macro Delay and Macro Pitch:

  • SPREAD distributes feedback tuning around the center lane.
  • DEVIATION adds repeatable lane-specific tuning and intensity variation.
  • SKEW biases the field from its first lane toward its last lane.
  • CENTER moves the neutral point of the spread.
  • GLIDE smooths relationship changes.

Containment

The internal loop caps its feedback gain, removes DC, monitors its own energy, and reduces regeneration when the loop becomes too dense. A final limiter keeps output finite at extreme settings. PANIC clears the loop immediately without changing the current parameter values.

These safeguards constrain digital output, but high monitoring gain can still be hazardous. Begin with low monitor level and raise FEEDBACK deliberately when using a live microphone.
