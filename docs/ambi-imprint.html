Ambi Imprint 64

s3g Ambi Imprint 64 applies a designed directional space response to a first-through-seventh-order ACN/SN3D field. It loads .s3gimprint files exported directly by the browser-based s3g-mc Imprint Sketch utility.

Signal Model

The 64-channel input remains available as the full-order dry field. The wet path samples the field through four or eight symmetric directional probes, routes each discrete early reflection toward its two nearest output probes, convolves the late response, and re-encodes the bank into the selected output order. This gives the response a deliberately lower spatial rank than the dry source, which suits reflected and diffuse energy while retaining project routing through a fixed 64-channel bus.

Kernel generation happens when an imprint is loaded, never in the audio callback. The file stores timing and spectral descriptors rather than sample-rate-specific FIR arrays, so the plugin builds kernels for the current host rate. Version 1 uses a four-second realtime response limit and reports 1024 samples of latency.

Imprint Sketch can describe connected wall, overhead, beneath, and stacked regions joined by rectangular, arched, circular, elliptical, slot, or irregular portals. Their path timing and directional loss are resolved into the loaded response profiles; the optional region graph drives the plugin's orbitable geometry display.

Low-Frequency and Gain Safety

The wet excitation path includes a fixed 25 Hz high-pass filter that removes DC and subsonic energy before the directional probes. Every first-order component uses the same filter response, and all subsequent safety gain is linked across the ambisonic field, so protection does not independently reshape channels or move the apparent image. The latency-aligned dry path remains unfiltered.

When an imprint is loaded, direct and sparse early-reflection taps receive a shared transfer-gain budget capped at +6 dB. Dense echo files therefore retain their relative timing, direction, and decay pattern without allowing hundreds of positive taps to create extreme low-frequency gain. A fast linked wet-field limiter provides a final guard before Mix and Output; user Output gain remains available after that guard.

Controls

  • ATLAS: loads one of nineteen bundled spaces spanning architecture, caves, caverns, tunnels, canyons, clearings, structured echoes, and abstract networks.
  • LOAD: selects an Imprint Sketch .s3gimprint file.
  • TOP, SIDE, and 3/4: set exact camera views. Drag the field to orbit and use the wheel or -/+ controls to zoom. Camera state is saved with the project.
  • ORD: selects first through seventh order; inactive higher channels are cleared.
  • MIX: blends the latency-aligned dry field with the convolved field.
  • FOC: changes how strongly first-order direction contributes to the response probes. At zero, the wet bank follows omnidirectional energy.
  • WID: scales higher orders in the re-encoded wet field.
  • OUT: output trim.
  • BYP: smoothly returns to the latency-aligned dry field.

Imprint Atlas

The bundled atlas contains nineteen deterministic, precomposed spaces. It includes conventional rooms, contrasting natural enclosures, open spaces, coupled passage systems, five geometry-derived echo designs, and mixed impossible networks. Atlas choices use the same .s3gimprint parser and DSP path as user files.

The repository also retains an editable s3g-imprint-sketch project for every atlas entry. The generation script rebuilds both forms from named seeds through the actual Imprint Sketch authoring engine and validates geometry, profiles, reflection values, and spectral bands before writing them.

Imprint Sketch Workflow

Open s3g-mc Imprint Sketch
- generate, mutate, design, or import an editable space project
- choose Export Imprint
- load the resulting .s3gimprint in Ambi Imprint 64

The imported descriptor is embedded in CLAP project state, so reopening a REAPER project does not depend on the original file remaining at the same path. The format reference documents the versioned interchange contract.

The plugin does not need to load a separate glTF file for display. It renders the primary polygon, ceiling profile, connected-region geometry, listener, and directional sources carried by .s3gimprint. Imprint Sketch glTF export remains available for interchange with dedicated 3D software.
