Bus Mixing / Routing

Bus tools treat a wide REAPER track as a collection of fixed-size source blocks. They mix or validate those blocks without assuming that the channels form an ambisonic field.

Group Matrix

Group Matrix 32/64 routes complete source blocks through a matrix. The 32-channel build supports stereo, 4-channel, and 8-channel groups; the 64-channel build supports 4-channel, 8-channel, and 16-channel groups. Every crosspoint moves the selected block together.

Node Bus Mixer

Node Bus Mixer 128 represents source blocks as nodes around a movable cursor. Node radius, focus, influence, and position turn cursor motion into coordinated source gains, providing a relationship-based alternative to a conventional matrix.

Routing Diagnostics

Passthrough Test 24ch is a fixed-width gain and routing check for REAPER pin assignments and CLAP bundle validation. Use it before diagnosing a more complex processor when a 24-channel insert appears to lose, swap, or bypass lanes.

When every block is a complete third-order ACN/SN3D field, use Ambi Field / Bus Effects so all sixteen channels in each field remain locked.
