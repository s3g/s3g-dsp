Sample / Loop Instruments

Sample and loop instruments load audio files as their source. The Loop family creates discrete eight-channel playback structures; Ambi Grain Processor treats a loaded ambisonic file as one locked encoded field.

Loop Family

  • Loop Processor: one loaded file distributed through an eight-channel loop instrument.
  • Multi Loop Processor: as many as four loaded files assigned across an eight-channel loop structure.

Ambisonic Granulation

Ambi Grain Processor loads 1OA, 2OA, or 3OA ACN/SN3D media. Grain time, direction, rate, and envelope remain shared across all active channels so the encoded image stays coherent.

Files and Project State

Loaded-media instruments depend on external source files. Keep those files with the project or in a stable media location, and confirm the source path after moving a session to another computer.
