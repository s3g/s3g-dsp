Mix / Pan

Mix / Pan covers tools that move, route, monitor, fold down, or place multichannel audio without assuming an ambisonic signal format. These plugins are about channel relationships, bus structure, speaker layouts, and direct panning.

Fold-Down / Monitoring

  • MC to Stereo Autogain: 128-channel input to true stereo output for auditioning multichannel tracks.
  • MC to Quad Autogain: 128-channel input to quad output ordered L, R, RB, LB.
  • Multichannel Meter 64: fixed-width passthrough meter for grid, spatial layout, and energy-over-time views.

Direct Panning / Layout

Direct panners write to speaker outputs. They do not create or require an ACN/SN3D ambisonic bus.

  • Layout Panner: direct source-to-speaker panning with speaker presets, custom layout design, and saved camera views.
  • DBAP / LBAP / VBAP Panner: alternate direct-panner gain laws using the same speaker-layout language.
  • Sub Crossover: layout-aware subwoofer crossover/send utility with up to eight subwoofer channels.
  • Array HPF / Array Delay / Array Trim 16/26/32/64: speaker-array calibration utilities for filtering, delay, trims, mutes, and polarity checks.

Bus Mixing / Routing

  • Group Matrix 32/64: matrix mixers for stereo, quad, octo, 4ch, 8ch, or 16ch groups.
  • Node Bus Mixer 128: node/cursor mixer for source blocks routed into one wide track.
  • Passthrough Test 24ch: fixed-width gain and routing check for REAPER and CLAP bundle validation.

Ambisonic Bus Variants

Use Ambi Field / Bus Processors when each source block is a complete 3OA ACN/SN3D feed.
