Ambi Ray Bilocation Encoder

s3g Ambi Ray Bilocation Encoder gives one mono source two simultaneous acoustic bodies. Space A and Space B each retain an independent direct path, early-reflection field, late response, movement history, and Doppler behavior before they are combined into a fixed 64-channel ACN/SN3D output.

Workflow

  1. Insert the plugin on a 64-channel track and route a mono source to its input.
  2. Choose a contrasting atlas pair or load independent .s3gray fields for A and B.
  3. Set PLACE, PERMEABILITY, MEMORY, and SEPARATION.
  4. Map source motion across both spaces and balance their room controls.
  5. Follow the encoder with an ambisonic decoder or meter.

Bilocation Topology

PLACE sends new source energy toward A or B. PERMEABILITY changes the transition from a narrow boundary into a broad region where both fields are excited. The send uses a smooth equal-power law, so the centre is a real dual presence rather than an interpolation between room descriptors.

MEMORY controls how quickly the return gate closes after a field stops receiving new energy. The abandoned field continues processing, allowing its early paths and late response to remain audible while the destination field materializes. At zero, the boundary is abrupt; longer settings carry a fading acoustic wake across it.

SEPARATION rotates the complete A and B ambisonic scenes in opposite directions. At zero degrees the fields share one directional frame. Larger settings unfold them toward opposing regions without changing either room's path lengths.

Position Mapping

The two large maps show both acoustic worlds at once. Drag either map to set shared normalized source or listener X/Y, select SRC or LIS at the central membrane, and use the Z controls below. Coordinates are mapped independently into each field's navigation bounds.

  • LINKED: the same normalized source trajectory inhabits both spaces.
  • MIRROR X or MIRROR Y: B reflects one axis of A's movement.
  • COUNTER: B inverts all three source axes, producing opposed motion.

The listener uses linked normalized coordinates in both spaces. MOVEMENT and DOPPLER retain the behavior of the single-space Ambi Ray Encoder.

Contrasting Pairs

PAIR loads two existing Ray Atlas fields plus a composed bilocation character. The eight pairs are Timber / Metal, Ice / Dry Stone, Open / Buried, Meadow / Impossible, Concrete / Water, Brick / Folded, Limestone / Conduit, and Flutter / Porous. They deliberately contrast scale, scatter, width, air damping, tail behavior, motion mapping, permeability, memory, and angular separation.

LOAD A and LOAD B independently accept version 1 .s3gray Ray Fields. Both imported fields are embedded in project state. Loading or replacing a pair constructs a complete dual runtime outside the audio callback.

Controls

ControlMeaning
DIRECT / EARLY / LATESet the shared component balance.
SIZE / SCATTER / WIDTH / AIRSet the independent character of each space.
ORDERSelects first through seventh order. Direct energy uses the selected order; both room fields use up to third order.
BYPASS ROOMRemoves early and late energy in both fields while preserving the two direct encodings.
OUTPUTSets final gain after a shared linked safety stage protects the combined ambisonic field.

Order and Routing

The output bus remains 64 channels wide. The selected order activates the corresponding ACN channels and clears the remainder.
