Multichannel Effects

Multichannel effects transform incoming REAPER track audio without assuming an ambisonic format. Use the other Multichannel sections for routing, fold-down, metering, and direct speaker panning. Use Ambisonic Effects when the channels form an ACN/SN3D field.

Sections

  • Topology Effects: multichannel effects whose lane relationships form a moving point network.
  • Macro Effects: compact delay, pitch, and shred interfaces with shared controls and lane relationship parameters.
  • Spectral / Buffer Effects: FFT-bin transformations and deliberately unstable buffer-window treatments.
  • Distributed Output Effects: stereo-input effects that distribute grains, delays, or taps across 16 ordinary output lanes.

Choose a Family

Start with topology effects when the relationship among input lanes should drive the result. Use a Macro effect for a smaller performance surface, Spectral / Buffer for transformations inside frequency or memory representations, and Distributed Output when a stereo source should emerge as a wide discrete-channel scene.

Routing

Effect widths describe ordinary input and output lanes, not an ambisonic order. Match the plugin build to the REAPER track width, and use the Multichannel monitoring, panning, or routing sections for the result.
