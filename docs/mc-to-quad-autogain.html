MC to Quad Autogain

s3g MC to Quad Autogain folds a multichannel REAPER track down to a four-channel quad output. It is a monitoring and audition utility for sessions where a quad speaker check is more useful than stereo, but a full decoder or speaker-layout workflow would be too much setup.

The plugin accepts up to 128 input channels and writes outputs 1-4 as L, R, RB, LB. It shares the same layout, rotation, width, 3D weighting, and autogain ideas as MC to Stereo Autogain, adapted to a quad field.

Output Order

The quad bus is intentionally ordered for straightforward REAPER routing:

Plugin outputSpeaker
1Left
2Right
3Right Back
4Left Back

Controls

ControlMeaning
INHow many plugin input channels are included in the fold-down.
WDTHQuad field width. Low values collapse inward; high values exaggerate the field.
ROTRotates ring and 3D layouts before they are folded to quad.
AGNDropdown menu for autogain: Off, Power/sqrt(N), or Energy sum.
OUTFinal quad output gain trim.
LAYDropdown menu for the layout model.
WGTControls how strongly the layout shape affects channel placement.
ATT3D attenuation for layouts with height and rear relationships.
DST3D distance spread for sphere, hemisphere, and cube layouts.

Layout Models

LAY uses the same layout names as the stereo fold-down: Ring projection, Linear left-right, Odd/even stereo, Center-out, Pair-preserving, Sphere projection, Hemisphere projection, and Cube projection. In the quad version, the channel positions are projected toward four speakers instead of a stereo pair.

AGN and LAY are discrete controls, so the GUI presents them as dropdown menus instead of continuous sliders.

REAPER Pins

The plugin has a four-channel output bus. For a clean fold-down insert, set the REAPER track channel count high enough for the source material, then route outputs 1-4 to the intended quad destination. If downstream channels still show unrelated audio, use REAPER's plugin pin connector to zero unmapped output channels.
