Ambisonic Order / Band Tool 64

s3g Ambisonic Order / Band Tool 64 is a fixed 64-in/64-out ACN/SN3D utility for trimming ambisonic orders and applying common order-weighting curves. It is useful before a decoder, before a stereo/head monitor, or between ambisonic effects.

input1OA..7OA
formatACN/SN3D
routing64 in / 64 out
weightsFlat/MaxRE/IP

Model

Ambisonic channels are grouped by order. Order 0 is the omnidirectional component. Higher orders carry progressively finer directional detail. This plugin scales whole order bands rather than individual channels, so the spatial format stays coherent.

The order map in the GUI shows the resulting order gain after the selected weighting preset, weighting amount, and per-order trims are combined.

Controls

  • ORD: active ambisonic order, from 1OA to 7OA. Channels above the selected order are muted.
  • WGT: weighting preset. Flat leaves orders level, MaxRE applies max-rE-style order weighting, In-phase applies a more conservative taper, and Custom uses only the order trims.
  • AMT: weighting amount. At 0%, only per-order trims are heard. At 100%, the selected preset is fully applied.
  • O0 through O7: per-order gain trims. These are useful for comparing localization, diffuse energy, and decoder stability.
  • OUT: final output gain.

Starting Points

  • Flat: best for checking whether a chain passes the original ambisonic signal without extra order shaping.
  • MaxRE: a useful default before speaker decoding when localization should remain clear without pushing the highest orders too hard.
  • In-phase: useful when the layout is irregular, the decoder feels too sharp, or a safer diffuse field is more important than tight localization.
  • Custom: use the order sliders as a manual spectral/spatial tilt across ambisonic order bands.

Routing

The plugin exposes one 64-channel input bus and one 64-channel output bus. Unused lanes are cleared. Keep it on a 64-channel REAPER track when using it in a 7OA-capable chain.
