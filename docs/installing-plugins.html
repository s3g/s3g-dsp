Installing Pre-release Plugins

These notes are for installing a packaged macOS CLAP zip from a GitHub pre-release. Plugin names, parameters, saved states, and included plugins may change.

The supported target is macOS + REAPER.

Requirements

  • A macOS pre-release zip from the s3g-dsp GitHub releases page.
  • REAPER.

Download

Pre-release builds are attached to GitHub releases when available. If no zip is available, use the Building From Source page.

Install

Unzip the pre-release package and copy the included .clap bundles to the user CLAP plugin folder:

~/Library/Audio/Plug-Ins/CLAP/

Replace older bundles with the same names if prompted. REAPER may need a plugin rescan after installation.

The package also includes a VOT Wavetables library and an Ambi Vox Demo Voicebank. Both can be stored anywhere. Load a wavetable WAV from Ambi VOT Encoder, or select the complete demo voicebank folder from the Ambi Vox Encoder PHRASE panel.

Current pre-release bundles are ad-hoc signed for bundle integrity but are not Apple notarized. macOS security policy may therefore require explicit approval for downloaded builds.

Included Groups

The current pre-release package installs 81 bundles. The groups below combine fixed-width variants so the inventory remains readable.

  • Multichannel Effects: Delay Processor, Wave Geometry Processor, Spectral Topology Processor, Macro Delay, Macro Pitch, Macro Shred, Spectral Spray 2ch/8ch, Buffer Processor, Shard Scatter, Orbit Delay, and Cascade Taps.
  • Monitoring / Fold-down: MC to Stereo/Quad Autogain and Multichannel Meter.
  • Direct Panning: Layout, DBAP, LBAP, and VBAP panners.
  • Bus Mixing / Routing: Group Matrix 32/64, Node Bus Mixer 128, and Passthrough Test 24ch.
  • Speaker-array Utilities: Sub Crossover and Array HPF/Delay/Trim 16/26/32/64.
  • Ambisonics: grouped as Encoders, Decoders, Effects, and Utilities, including the Point/Cloud/Terrain/Path/Ray/Ray Bilocation/VOT/Vox/Wave Terrain/Wind/Water/Pulsar/Stochastic/Wrangler encoder families, speaker and monitoring decoders, 3OAFX and field effects, interchange formats, and visualization.
  • Instruments: Fault, Loop Processor, Multi Loop Processor, Ambi VOT Encoder, Ambi Vox Encoder, Ambi Wave Terrain Encoder, Ambi Wind Encoder, Ambi Water Encoder, Ambi Pulsar Encoder, Ambi Stochastic Encoder, Ambi Wrangler Encoder, and Ambi Grain Processor.

REAPER Setup

After installing, open REAPER and rescan CLAP plugins if the new builds do not appear automatically. Search for s3g in the FX browser.

For multichannel sessions, set the track channel count before inserting the plugin.

For true stereo-output plugins on wide REAPER tracks, including s3g MC to Stereo Autogain, s3g Ambi Stereo Decoder, and s3g Ambi Head Decoder, open REAPER's plugin pin connector and enable Zero out unmapped output channels if a following meter still shows channels above 2. These CLAP plugins expose stereo output buses, but REAPER can still pass unmapped track channels around the effect unless this option is enabled.

s3g MC to Quad Autogain exposes a four-channel output bus ordered L, R, RB, LB. s3g Shard Scatter, s3g Orbit Delay, and s3g Cascade Taps expose sixteen output channels, so set the REAPER track channel count and plugin pins before auditioning those processors.

Fault is a zero-audio-input instrument with a fixed eight-channel output. Set the REAPER track to eight channels before inserting it; no audio input routing is required. Its FREE mode needs no MIDI, while MIDI mode accepts note input from the track.

For mix, route, pan, and monitoring tools, see Multichannel. For ambisonics-specific tools, see Ambisonics. For the larger REAPER package workflow, see the s3g-mc 3OAFX Send Return Controller guide.

Cleanup

If older pre-release plugins appear in REAPER, remove stale bundles from the user CLAP folder and rescan:

~/Library/Audio/Plug-Ins/CLAP/

Build folders can also contain stale artifacts after plugin renames. Reconfigure and rebuild with the current preset if needed.
