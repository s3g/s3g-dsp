Ambi Rotate 64

s3g Ambi Rotate 64 is a fixed 64-in/64-out ACN/SN3D ambisonic utility for rotating a field before decoding, analysis, or further processing. It supports first through seventh order.

Use

Place Rotate between an ambisonic source and a decoder or visualizer when the field needs to turn without changing the speaker layout. For example, it can rotate a captured or generated scene before the Speaker Decoder, Stereo Decoder, Head Decoder, or Energy Visualizer.

The rotation uses a precomputed spherical projection matrix for the selected order. It is designed for practical field-orientation work in REAPER while keeping a stable 64-channel bus.

Controls

ControlMeaning
ORDActive ambisonic order, from 1OA to 7OA. Higher orders use more channels.
YAWHorizontal rotation. This is the usual azimuth rotation control.
PITVertical pitch/elevation rotation.
ROLRoll rotation around the front axis.
WIDOrder-dependent width/sharpness trim. 1.00 is normal; lower values pull high-order spatial detail inward; values above 1.00 exaggerate high-order detail.
OUTFinal output gain.

Routing

The plugin exposes one 64-channel input bus and one 64-channel output bus. Channels above the selected order are cleared so downstream tools do not see stale higher-order lanes.

Use the same order setting through nearby ambisonic utilities unless you intentionally want to limit the field. A 3OA source can run through a 7OA-width plugin safely, but only the channels present in the source carry meaningful signal.
