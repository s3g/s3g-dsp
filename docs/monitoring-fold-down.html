Monitoring / Fold-down

Monitoring tools audition or inspect wide multichannel tracks without changing the source routing model. Use a fold-down for an ordinary stereo or quad monitor path and a passthrough meter when the original channel layout must remain available downstream.

Fold-down Plugins

  • MC to Stereo Autogain: accepts as many as 128 input channels and produces a true stereo output with automatic energy management and 3D placement controls.
  • MC to Quad Autogain: produces L, R, RB, and LB quad output from a wide source track.

Metering

Multichannel Meter 64 is a fixed-width passthrough meter with channel-grid, spatial-layout, and energy-over-time views. It is the better choice when the monitor display must not collapse the bus.

Host Routing

The stereo and quad tools expose narrower output buses. In REAPER, inspect the plugin pin connector and enable Zero out unmapped output channels when unused track channels would otherwise pass around a fold-down plugin.
