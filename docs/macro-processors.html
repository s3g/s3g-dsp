Macro Effects

Macro effects expose a compact engine plus a small set of lane-relationship controls. They are intended for fast multichannel shaping when a full topology editor would be unnecessary.

Plugins

  • Macro Delay 8ch/24ch: shared delay, feedback, filtering, and mix controls with lane time, feedback, and level relationships.
  • Macro Pitch 8ch/24ch: shared pitch-shift controls with coordinated lane offsets and movement.
  • Macro Shred Mono/8ch/24ch: pressure, wavefolding, and bounded resonant feedback for voice, microphone, and general audio input.

Relationship Controls

The multichannel builds describe how lanes differ around a shared setting instead of publishing a complete parameter bank per channel. Spread, slope, alternating, and motion controls create repeatable relationships with fewer automation lanes.

Use a topology effect when the result should respond to an editable point network. Use a Macro effect when the main goal is a compact shared sound with controlled lane variation.

Plugin Widths

Insert the 8-channel or 24-channel build on a matching REAPER track. Macro Shred also includes a mono build for direct voice and microphone processing.
